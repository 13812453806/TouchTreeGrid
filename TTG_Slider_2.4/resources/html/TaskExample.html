Task List - Accordion Example
 

>Touch category row to expand/collapse tasks. Accordion SingleExpand mode supported. Opacity animation added to this example.

 

>Expand/Collapse buttons (all rows)

 

>Expand/Collapse Footer toolbar auto-added by TouchTreeGrid component

 

>Every display aspect easily configured with CSS

 

>See Census example for displays customized differently between Phones and Tablets

 

CODE TO IMPLEMENT:

(simply define Tree Store, link to TouchTreeGrid component and set configs)

 

{

 xtype: 'touchtreegrid',

 store: 'TaskAccordionStore',

 columns: [

{

   dataIndex: 'text',

   width: '35%',

   style: ' text-align: left; font-size: 1.5em;',

   categStyle: 'font-weight: bold; font-size: 2em; padding-top: .2em; text-align: left; color: blue;',

}

 ],

 listItemId: 'firstexamplelist',

 includeFooterLevels: false,

 includeHeader: false,

 singleExpand: true,

 defaultCollapseLevel: 1,

 cls: [

 'x-touchtreegrid-list',

 'x-touchtreegrid-list-accordion'

 ],

 itemId: 'firstexample'

}
