Locked Columns Grid simulating EXTJS Big Data example
 

>Class Name: TouchTreeGrid.view.BigDataCont

 

>Standard Grid example (simplGrid=true) with custom thumb to resize locked columns width (implemented for phones only). See help for Standard Grid #1 example for recommended configs.

 

>Also demonstrates data grouping and grouped column headers. Default thumb overridden to look like folder tab. headerTplOverride used to implement grouped column headers.

 

>Refer to TouchTreeGrid.view.BigDataCont initialize() method for how to only enable thumb and to resize locked grid width for phones. In this case list width is wider than grid width for phones to prevent data wrapping.
