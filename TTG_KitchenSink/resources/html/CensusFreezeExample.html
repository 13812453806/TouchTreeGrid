Freeze Column Example
 

>Class Name: TouchTreeGrid.view.CensusFreezeCont

 

>Horizontal scrolling with Freeze Column for large 600 data row with 28 data elements per row Tree Grid example. Default thumb configuration used for this example

 

>See help for Standard Grid #1 example for recommended configs

 

This example implements pinch event to allow user to resize grid to 3 pre-defined configurations (also press buttons to see same for non-mobile devices). Refer to TouchTreeGrid.view.CensusFreezeCont initialize() method for how to customize expandAll/collapseAll buttons to reduce used space when implemented for locked columns. Locked grid minWidth/maxWidth defined for this example and used to control how far thumb can be dragged.
