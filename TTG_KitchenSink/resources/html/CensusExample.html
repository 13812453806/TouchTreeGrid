Census Example
 

>Class Name: TouchTreeGrid.view.CensusMaineContainer

 

>Large 600 data row with 28 data elements per row example. Observe fast scrolling performance for fully expanded rows.

 

>Initially collapsed to Level 2

 

>Default category and expand/collapse button colors.

 

>Arrow width 2% for Tablets and 4% for Phones using arrowPctWidth config

 

>Rotate to landscape (phones only) to expand display with additional columns ... rotate back to restore

 

>Tablets will display even more columns and # of displayed columns will automatically change when Tablet is rotated between Portrait and Landscape

 

>Touch to right of expanded items to see full detail of selected row (28 Census categories)

 

>Standard Sencha Touch Disclose icon used and included on both Collapsed and Expanded rows. Compare touch sensitivity of this icon with smaller icon on Project example

 

>TouchTreeGrid componet provides column formatting functions for comma-delimitted, currency, percents. Custom renderering functions are easy to implement (example different colors for positive vs. negative values)
