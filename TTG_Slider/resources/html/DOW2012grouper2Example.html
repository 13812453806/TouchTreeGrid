DOW 2012 Grouper#2 Example
 

>Variation of Grouper #1 example where TreeGrid used for grouped rows. "categColumns" array allows category rows to be styled independent of detail rows.

 

>CONFIG** categColumns: {

 [{

 dataIndex: 'grouper',

 width: '90%',

 categStyle: 'text-align: left; color: #006083; font-weight: bolder; font-size: 1.2em;'

 }]
