DOW 2012 Horizontal Scrolling Example
 

>Custom multi-row header. Customized column shading for sorted column

 

>Also implements Vertical lines and color renderers for positive/negative values

 

>Achieved by defining column array widths in pixels (or em's) vs. percents and making parent container scrollable horizontally

 

>Other significant configs:

- Parent container has layout: 'hbox'

- height, width and minWidth specified

 

NOTE: Horiz scrolling disabled in this example if screen width > 576 pixels since scrolling not needed to fit grid (test on phone portrait)
